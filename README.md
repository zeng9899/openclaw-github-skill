# GitHub Trending + Feishu Skill

获取GitHub热门AI项目数据，生成详细简报并发送到飞书。

## 功能

- ✅ 自动获取GitHub热门AI项目
- ✅ 生成详细简报（包含Stars、Fork、描述、技术栈、链接等）
- ✅ 一键发送到飞书
- ✅ 无需RoxyBrowser
- ✅ 无需GitHub Token
- ✅ 使用GitHub公开API（免费）

## 使用方法

直接对AI助手说：
- "查一下github热门AI项目，做个简报发给我"
- "生成word简报"

## 技术实现

- 使用 GitHub 公开API 获取数据
- 使用飞书消息接口发送报告
- 无需任何API密钥

## 依赖

- OpenClaw 框架
- 网络访问（获取GitHub数据）
